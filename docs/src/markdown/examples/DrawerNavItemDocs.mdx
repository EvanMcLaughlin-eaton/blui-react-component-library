# Nested Drawer Items

Navigation items can be nested to allow for different levels of navigation. To create a nested navigation item, simply place a `<DrawerNavItem />` inside of another. The `<DrawerNavItem />` allows nested navigation up to 3 levels deep.

<DrawerNestedDrawerItems />

# Selected Navigation Items

activeItem prop of Drawer should be updated with itemId of selected `<DrawerNavItem />`. The active item's highlighted shape can be adjusted via the [activeItemBackgroundShape] input to be either "square" (default) or "round".
