# Basic Info List Item

The `<InfoListItem>` is intended to be used in List Views. A minimal example accepts a `title` prop.

<BasicInfoListItem />

# Dense Info List Item

The `<InfoListItem>` accepts an optional `dense` prop to make the row skinnier.

<DenseInfoListItem />

# Info List Item with Subtitle

The `<InfoListItem>` accepts an optional `subtitle` prop that appears on the second line.

<InfoListItemWithSubtitle />

# Info List Item with Icon

The `<InfoListItem>` accepts an optional `icon` prop that will appear to the left.

<InfoListItemWithIcon />

# Info List Item with Status

The `<InfoListItem>` accepts an optional `statusColor` prop that adds a status stripe to the left side of the row. This can be paired with the `avatar` prop to add a padding and color around the `icon`.

<InfoListItemWithStatusColor />

# Info List Item with Left Content

The `<InfoListItem>` accepts an optional `leftComponent` prop which renders between `icon` and `title`.

<InfoListItemWithLeftComponent />

# Info List Item with Right Content

The `<InfoListItem>` accepts an optional `rightComponent` prop which renders at the right edge of the item.

<InfoListItemWithRightComponent />

# Info List Item in a List

The `<InfoListItem>` typically appear in a list which contains multiple rows of data.
