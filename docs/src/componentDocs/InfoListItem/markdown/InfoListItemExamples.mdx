# Basic InfoListItem

The `<InfoListItem>` is intended to be used in List Views. A minimal example accepts a `title` content.

<BasicInfoListItem />

# Dense InfoListItem

The `<InfoListItem>` accepts an optional `dense` input to make the row skinnier.

<DenseInfoListItem />

# InfoListItem with Subtitle

The `<InfoListItem>` accepts an optional `subtitle` content that appears on the second line.

<InfoListItemWithSubtitle />

# InfoListItem with Icon

The `<InfoListItem>` accepts an optional `icon` content that will appear to the left.

<InfoListItemWithIcon />

# InfoListItem with Status

The `<InfoListItem>` accepts an optional `statusColor` input that adds a status stripe to the left side of the row. This can be paired with the `avatar` input to add a padding and color around the `icon`.

<InfoListItemWithStatusColor />

# InfoListItem with Left Content

The `<InfoListItem>` accepts an optional `leftComponent` which renders between a `icon` and `title`.
