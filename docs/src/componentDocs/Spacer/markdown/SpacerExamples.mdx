# Spacer with Flex

The `<spacer>` adds space between elements. It accepts a flex prop that determines how much relative space to consume.
For a more contextual example, see how a `<spacer>` can be used to separate multiple [`<DrawerNavGroup>`](/components/drawer/drawer-nav-group/examples).
