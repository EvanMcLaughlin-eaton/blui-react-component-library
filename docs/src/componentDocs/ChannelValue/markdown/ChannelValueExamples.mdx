The `<ChannelValue>` accepts a `value` and a `units` props to display metric data.

# Channel Value with Prefix

The `<ChannelValue>` accepts an optional `prefix` prop that will show the `units` before its `value`.

<ChannelValueWithPrefix />

# Channel Value with Different Units

The `<ChannelValue>` will hide the whitespace by default between a `value` and its `units` when the `units` '%', '°F', '°C', '°' or '$' are used. This behavior can be overridden via the optional `unitSpace` prop.
