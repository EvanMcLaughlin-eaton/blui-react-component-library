# Basic Drawer Header

A `<DrawerHeader />` appears at the top of a `<Drawer />` and can be used to provide context to an application through a title input or a branding image.

<BasicDrawerHeader />

# Drawer Header with Subtitle

A `<DrawerHeader />` accepts a `title` and a `subtitle` prop.

<DrawerHeaderWithSubtitle />

# Drawer Header with Icon

A `<DrawerHeader />` accepts a `icon` content child which can be used to manage the `<Drawer />` opened state.

<DrawerHeaderWithIcon />

# Drawer Header with Custom Content

Custom header content can also be passed as a `titleContent` prop.
