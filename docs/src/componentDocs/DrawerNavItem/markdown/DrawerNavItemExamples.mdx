# Nested Drawer Nav Items

Navigation items can be nested to allow for different levels of navigation. To create a nested navigation item, simply place a `<DrawerNavItem />` inside of another. The `<DrawerNavItem />` allows nested navigation up to 3 levels deep.

<DrawerNavItemNestedItems />

# Selected Navigation Items

`activeItem` prop of Drawer should be updated with itemID of selected `<DrawerNavItem />`. The active item's highlighted shape can be adjusted via the `activeItemBackgroundShape` prop to be either "square" (default) or "round".
