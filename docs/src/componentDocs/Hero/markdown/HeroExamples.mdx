# Basic Hero

The `<Hero />` accepts a `label` prop and the `icon` prop will accept any valid component.

<BasicHero />

# Hero with Value and Units

The value section of the `<Hero />` utilizes a `<ChannelValue />` component.
To display a single simple value, the information can be passed using `ChannelValueProps`. It is commonly used to display metric data.

<HeroWithValueAndUnits />

# Hero with Channel Value Children

Users can pass in `<ChannelValue>` components as children and they will be displayed inline.

<HeroWithChannelValue />

# Hero with a Secondary Icon

The `<Hero />` accepts an optional secondary icon which we can pass through `ChannelValueProps`.

<HeroWithSecondaryIcon />

# Heroes with a Hero Banner

The `<HeroBanner />` component is a simple wrapper component that accepts `<Hero />` components as its children.

<HeroBanner />

# Hero with Icon Size and Image Types

The `<Hero />` supports multiple image types and has an `iconSize` prop to scale.

<HeroBannerWithIconSize />
